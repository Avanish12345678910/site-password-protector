üöÄ Chrome Installation

  1. Open Chrome and navigate to chrome://extensions/
  2. Enable "Developer mode" toggle in the top-right corner
  3. Click the "Load unpacked" button
  4. Browse to and select the extension folder
  5. The extension icon will appear in your browser toolbar!

ü¶ä Firefox Installation

  1. Swap the manifest files for Firefox compatibility:
    mv manifest.json manifest_chrome.json mv manifest_firefox.json manifest.json
  2. Open Firefox and go to about:debugging#/runtime/this-firefox
  3. Click "Load Temporary Add-on"
  4. Select the manifest.json file from the extension folder
  5. Extension loaded! (Note: temporary installation, removed on browser close)

üìñ How to Use

  1. Add a Protected Site: Click the extension icon, enter a domain (e.g., facebook.com) and password
  2. Visit the Site: Navigate to the protected website in your browser
  3. Enter Password: You'll see a beautiful password prompt overlay
  4. Access Granted: Enter the correct password to unlock for 30 minutes
  5. Manage Sites: Click the extension icon to view, edit, or remove protected sites

üí° Tips & Tricks

  • Enter domains without http://, https://, or www
  • Use the "Clear Session" button to require immediate re-authentication
  • Passwords are case-sensitive, so choose carefully!
  • Session duration can be customized in background.js
  • Perfect for shared computers or parental controls
